Recommended Software

    Research, Mapping, and Data Analysis

  • RStudio – Working environment for R, ideal for statistical computing and graphics.
  • QGIS – An open-source Geographic Information System for spatial analysis and mapping. Good for finishing maps before publication.
  • Zotero – Reference manager. Use with Zotfile for syncing and to get around storage limitations.
  • Miro – An online tool for creating beautiful mindmaps and flowcharts.
  • ColorSpace – An online tool for creating colour palettes.

    • Media Tools

  • Paint.NET – For image manipulation.
  • Audacity – For audio manipulation.
  • MusicBee – For playing local music files.
  • VLC Media Player – For playing local video files.

    • File Management

  • FreeFileSync – Synchronises files and folders across devices. Awesome for keeping work synchronised across multiple filesystem: I used this to keep my PhD synchronised across my home PC, portable SSD, and onedrive-linked work laptop.
  • VeraCrypt – Encryption tool.
  • 7-Zip – File compression tool.
  • Calibre – E-book manager.
  • WinDirStat – Digital storage analyser.
  • AMD RAMDisk – Allows system RAM to function as super-fast storage for temporary files. 4GB limit on free version.
  • Open Hardware Monitor – System monitoring.

    • Internet & Security

  • Mozilla Firefox – A secure, privacy-focused web browser with customisation options.
  • Bitwarden – An open-source password manager for securing credentials.
  • uBlock Origin – A powerful ad-blocker.
  • SponsorBlock – A browser extension that skips sponsored segments in videos.
  • SwiftSelectionSearch – A browser extension allowing quick searches by highlighting text.

    • Tools

  • Outlook Google Calendar Sync – Syncs Google and Outlook calendars.
  • Revo Uninstaller – Uninstalls software and cleans up leftover files.
  • Unified Remote – Turns a smartphone into a PC remote control.
  • 12ft.io – Paywall bypasser.
  • Rainy Mood – Rain noise generator.
  • Free Lunch – Awesome list of free software.

    • Miscellaneous

  • Cycle.Travel – Bicycle route planner. Finds quiet roads.
  • Ninety 180 – Good visualisation tool for post-Brexit Brits. Calculate how many days you can remain in the EU.
